# Essential command

### Cloning git repo
~~~
git clone <Repo> 
~~~

### getting directory all file
~~~
ls 
dir
~~~

### change directory
~~~
cd  <folderName>
~~~
### to create a file
~~~
touch <filename>
~~~

### to create a folder
~~~
mkdir <folderName>
~~~

### to back to parents directory
~~~
cd ..
~~~

### to clear screen
~~~
cls - windows
clear - linux
~~~
